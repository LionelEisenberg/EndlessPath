Slot anatomy

  • Icon — the consumable's resource icon. Pulled from ConsumableDefinitionData.icon.
  • Stack count (top-right badge) — how many of this consumable the player has. When it reaches 0 the slot empties.
  • Hotkey 1 2 3 (bottom-right) — keyboard shortcut for that slot. Same shape as the ability hotkeys (Q W E R).
  • Cooldown sweep (slot 2) — dark radial overlay that recedes as ConsumableCooldownTracker.remaining(def) drops to 0. Center shows the integer seconds remaining.
  • Empty slot (slot 3) — dashed border, no icon, hint text. Clicking opens the inventory's Consumables tab focused on the equip-slot grid (see other mockup).

What's not shown

  • No "Use" button — using a consumable is keyboard-only (hotkey) or click-on-slot. No menu walk required.
  • No tooltip — hover behavior is a separate design pass (we'd reuse the existing item tooltip panel).
  • Slot count is illustrative — actual number (3? 4?) is a tuning decision for the equip-slot spec.
